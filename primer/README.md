# Sunodo Primer - JavaScript DApp Template

This repo contain a prime number generator. Whenever the input is a string, it will format the string and return it back to the terminal.

This is a template for JavaScript Cartesi DApps. It uses node to execute the backend application.
The application entrypoint is the `src/index.js` file. It is bundled with [esbuild](https://esbuild.github.io), but any bundler can be used.

## Build the project
```
sunodo build
```
## Backend service
To run the backend service
```
sunodo run
```
## Usage
To run this code, run:
```
sunodo send generic
```
